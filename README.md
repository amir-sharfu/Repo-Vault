## How to Navigate

**Browse by category** → open any `.md` file (`ai-ml.md`, `devtools.md`, etc.) for a quick-reference table of every repo in that category.

**Get the full picture on a repo** → click the **notes** link to open its `repo-insider/` deep-dive — diagram, problem/solution, features, use cases, and when to use it.

**Explore a website** → click the **notes** link in any website row to open its `notes/` file — same structure, focused on the site.

**Navigate all deep-dives at once** → open [`repo-insider/README.md`](./repo-insider/README.md) for a categorised index of all 42 deep-dives.
