## How to Use

**Repos** — Each category file contains a table with:
- **Repo** — Link to the repository
- **Stars** — Approximate star count at time of adding
- **Description** — What it does
- **Added** — Date added

**Websites** — The `websites.md` file contains a table with:
- **Site** — Site name
- **URL** — Direct link
- **Description** — What it offers
- **Tags** — Quick-search labels
- **Added** — Date added
