## How to Navigate

**Browse by category** → open any `.md` file (`ai-ml.md`, `security.md`, etc.) for a quick-reference table of every repo in that category.

**Get the full picture on a repo** → click the **notes** link in the table to open its `repo-insider/` deep-dive — you'll find a diagram, a problem/solution breakdown, core features, real-world use cases, and a "when to use it" guide.

**Explore a website** → click the **notes** link in any website row to open its `notes/` file — same structure: what it is, why it matters, key features.
